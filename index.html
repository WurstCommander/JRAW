Java Reddit API Wrapper

Examples

Note: The object redditClient is instantiated with new RedditClient("MY-USER-AGENT")

Log in to your account

LoggedInAccount me = redditClient.login("MY-USERNAME", "MY-PASSWORD");

Save every submission on the front page

LoggedInAccount me = redditClient.login("MY-USERNAME", "MY-PASSWORD");

SimplePaginator frontPage = reddit.getFrontPage();
Listing<Submission> submissions = frontPage.next();

for (Submission submission : submissions.getChildren()) {
    me.save(submission);
}

Get a new captcha

Captcha captcha = redditClient.getNewCaptcha();
// do something with captcha.getImageStream()

Get a user's information

Account account = redditClient.getUser("USERNAME");
